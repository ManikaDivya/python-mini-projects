# 📊 Sales Data Analysis Project

## 🔹 Overview
This project analyzes a large sales dataset to find useful business insights.

## 🔹 Tools Used
- Python
- NumPy
- Pandas

## 🔹 Dataset
Superstore sales dataset containing order details, sales, profit, and customer information.

## 🔹 Key Analysis Performed
- Data cleaning and preprocessing
- Category-wise analysis
- Region-wise analysis
- Monthly sales trend
- Top products analysis

## 🔹 Key Insights
- Technology category has the highest sales and profit
- West region is the top-performing region
- South region has the lowest performance
- Sales peak in November indicating seasonal demand
- Canon imageCLASS 2200 Advanced Copier is the top-selling product

## 🔹 Conclusion
This project helps understand sales patterns and supports better business decisions.
